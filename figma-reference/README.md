# Figma Reference Pack

Arquivos gerados para virar uma base de edicao rapida no Figma.

## Como usar

1. Abra um arquivo novo no Figma.
2. Crie uma pagina chamada `TikTok Analytics Reference`.
3. Arraste estes PNGs para dentro do canvas.
4. Use `dashboard-desktop-full.png` e `dashboard-mobile-full.png` como referencia principal.
5. Use os PNGs por bloco para comentar, reorganizar e marcar o que precisa mudar sem poluir a tela inteira.

## Arquivos

- `dashboard-desktop-full.png`: tela completa desktop
- `dashboard-mobile-full.png`: tela completa mobile
- `dashboard-hero.png`: cabecalho e hero
- `dashboard-metrics.png`: cards de metricas
- `dashboard-growth.png`: bloco de crescimento/grafico
- `dashboard-diagnostics.png`: bloco de diagnosticos
- `dashboard-filters.png`: filtros e ordenacao
- `dashboard-posts.png`: tabela/lista de posts
- `dashboard-coverage.png`: cobertura de dados

## Fluxo recomendado

- Duplique os blocos que voce quiser alterar.
- Desenhe por cima com retangulos, textos e setas.
- Se quiser, renomeie os frames por prioridade:
  - `P1 - mudar hierarquia`
  - `P1 - simplificar copy`
  - `P2 - novo card`
  - `P2 - nova secao`

Quando voce terminar de anotar no Figma, me manda o link ou screenshots das mudancas e eu comeco a implementar no codigo.
